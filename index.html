Get Started in Minutes

1

Install the Package

Add our library to your project using npm or yarn.

npm install yahoo-fantasy-sports-api
2

Get Yahoo API Credentials

Create an app in the Yahoo Developer Console and get your Client ID and Secret.

// Visit: https://developer.yahoo.com/apps/
// Create new app ‚Üí Fantasy Sports API ‚Üí Get credentials
3

Initialize and Start Coding

Import the library and start accessing your fantasy data immediately.

import YahooFantasy from 'yahoo-fantasy-sports-api';

const yf = new YahooFantasy({
  clientId: 'your_client_id',
  clientSecret: 'your_client_secret',
  redirectUri: 'http://localhost:3000/auth/callback'
});

// Get user's leagues
const leagues = await yf.getLeagues();
console.log('My Fantasy Leagues:', leagues);

Code Examples

See how easy it is to work with fantasy sports data 

// Get league information
const league = await yf.getLeague('414.l.123456');
console.log(`League: ${league.name} (${league.season})`);

// Get all teams in a league
const teams = await yf.getTeams('414.l.123456');
teams.forEach(team => {
    console.log(`${team.name} - ${team.wins}-${team.losses}`);
});

// Get player information
const player = await yf.getPlayer('nfl.p.24171');
console.log(`${player.name} - ${player.position} (${player.team})`);

// Get matchup data
const matchups = await yf.getMatchups('414.l.123456', 1);
matchups.forEach(matchup => {
    console.log(`${matchup.teams[0].name} vs ${matchup.teams[1].name}`);
});
